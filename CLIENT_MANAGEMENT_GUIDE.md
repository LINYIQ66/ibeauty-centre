## 🔧 技术架构

### 数据库表结构

**clients 表：**
```sql
CREATE TABLE clients (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    phone VARCHAR(20) UNIQUE NOT NULL,
    gender VARCHAR(20),
    email VARCHAR(100),
    birthday DATE,
    address TEXT,
    notes TEXT,
    membership_level VARCHAR(50) DEFAULT 'regular',
    total_spent DECIMAL(10,2) DEFAULT 0,
    total_visits INTEGER DEFAULT 0,
    last_visit DATE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**client_tags 表：**
```sql
CREATE TABLE client_tags (
    id SERIAL PRIMARY KEY,
    name VARCHAR(50) UNIQUE NOT NULL,
    color VARCHAR(7) DEFAULT '#3b82f6',
    description TEXT
);
```

**client_tag_map 表：**
```sql
CREATE TABLE client_tag_map (
    client_id INTEGER REFERENCES clients(id),
    tag_id INTEGER REFERENCES client_tags(id),
    PRIMARY KEY (client_id, tag_id)
);
```

**client_transactions 表：**
```sql
CREATE TABLE client_transactions (
    id SERIAL PRIMARY KEY,
    client_id INTEGER REFERENCES clients(id),
    appointment_id INTEGER,
    service_id INTEGER,
    employee_id INTEGER,
    amount DECIMAL(10,2),
    transaction_type VARCHAR(50),
    payment_method VARCHAR(50),
    points_earned INTEGER,
    transaction_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 自动触发器

**功能：** 预约创建时自动更新客户统计

```sql
CREATE TRIGGER update_client_stats_trigger
AFTER INSERT ON appointments
FOR EACH ROW
EXECUTE FUNCTION update_client_stats();
```

**触发器逻辑：**
1. 更新 `total_visits`（消费次数 +1）
2. 更新 `total_spent`（总消费累加）
3. 更新 `last_visit`（最后到访日期）
4. 插入 `client_transactions`（交易记录）

## 📚 API 文档

### 客户管理端点

**GET /api/clients**
- 获取所有客户列表
- 包含消费统计和标签
- 响应：`{clients: [...]}`

**POST /api/clients**
- 创建新客户
- 请求体：`{name, phone, email, membership_level, tags}`
- 响应：`{success: true, client: {...}}`

**PATCH /api/clients/{id}**
- 更新客户信息
- 请求体：要更新的字段
- 响应：`{success: true, client: {...}}`

**GET /api/clients/{id}/transactions**
- 获取客户交易记录
- 响应：`{transactions: [...]}`

**GET /api/client-tags**
- 获取所有标签
- 响应：`{tags: [...]}`
